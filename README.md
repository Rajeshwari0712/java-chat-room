# 💬 Java Multi-User Chat Room

A desktop-based **Multi-User Chat Room Application** developed using **Java Swing** and **Multithreading**. The application simulates multiple users communicating in real time within a shared chat environment through an interactive graphical user interface.

---

## 📌 Features

* 💬 Multi-user chat simulation
* ⏰ Timestamped messages
* 😊 Emoji support (`:)`, `:(`, `:D`, `<3`)
* 🎨 Customizable chat text colors
* 🔤 Adjustable font size
* 🎨 Background color customization
* 👥 Live user list
* 🗂️ Tab-based interface (Chat, Settings, About)
* 🧹 Clear chat option
* 🚪 Exit functionality

---

## 🛠️ Technologies Used

* Java
* Java Swing
* Multithreading
* Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone this repository.

```bash
git clone https://github.com/Rajeshwari0712/java-chat-room.git
```

2. Open the project in **Eclipse** or **IntelliJ IDEA**.

3. Make sure **Java JDK 21** (or a compatible version) is installed.

4. Run **FunChatRoom.java**.

5. Two chat windows will open, allowing users to exchange messages.
